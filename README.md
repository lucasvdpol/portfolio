Portfolio Website

Dit is mijn persoonlijke portfolio-website, ontwikkeld met HTML5 en CSS3. De site fungeert als mijn digitale visitekaartje en toont mijn projecten, vaardigheden en achtergrond als software developer.
Structuur
-index.html: De homepage met een overzicht van mijn profiel en projecten.
-styles.css: Bevat de stylesheet voor de layout en vormgeving.
-assets/: Bevat afbeeldingen en andere media (zoals screenshots van mijn projecten).

Gebruikte technieken
-HTML5: Voor de semantische structuur van de pagina's.
-CSS3: Voor de vormgeving, responsive design en animaties.
-Hosting: Gehost via GitHub Pages (live te bekijken op: https://lucasvdpol.github.io/portfolio/).

Lokale installatie

Omdat dit een statische website is, heb je geen ingewikkelde server-setup nodig:
-Clone deze repository naar je lokale machine.
-Open het bestand index.html in je webbrowser.
-(Optioneel) Gebruik de "Live Server" extensie in VS Code voor een betere ontwikkelervaring.
